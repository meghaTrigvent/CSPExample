CSP Intro

Defining restrictions on the scripts, styles, and other resources an extension can use/load.

In Manifest V3, all of your extension's logic must be part of the extension package. You can no longer load and execute remotely hosted files.

Examples include:

JavaScript files pulled from the developer's server.

Any library hosted on a CDN.

Bundled third-party libraries that dynamically fetch remote hosted code.

How to set CSP rules

  • Define the content_security_policy key in your manifest.json file.
  • Set the CSP rules according to your extension's requirements.
  • Ensure your CSP rules follow a valid CSP syntax, including directives like script-src, style-src, and others.
